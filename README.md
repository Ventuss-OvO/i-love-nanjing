## 项目介绍

本项目收集了李志的全部公开发行的音乐作品，拥有完整的ID3信息，并提供 AAC 和 Apple Lossless 两种格式的高质量音频文件。

本人只是普通乐迷，收集这些作品只是出于个人兴趣，本项目仅供个人欣赏，请勿商用，所有版权归原作者所有。
